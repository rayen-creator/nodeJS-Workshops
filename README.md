
# Nodejs workshops

expressjs apps in one repo 

## Usage

1-clone the repo 

```
git clone https://github.com/rayen-creator/nodeJS-Workshops
```
2- choose whatever app to run

```
cd <app_name>
```
3-Install npm packages

 ```sh
 yarn install
 ```
or

```sh
 npm install
```

3- run app

```sh
 nodemon server.js
```
